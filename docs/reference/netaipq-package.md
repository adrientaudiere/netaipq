# netaipq: Networks and Machine Learning for 'phyloseq' Objects

Network, graph, causal-inference, and machine-learning tools for
'phyloseq' objects within the 'pqverse' ecosystem. Provides genuinely
new analysis methods (co-occurrence and association networks,
bipartite-network visualisation, supervised classification,
indicator-species and differential-abundance analyses) that go beyond
simple 'ggplot2' wrappers. 'netaipq' is the home for any item adding a
brand-new machine-learning, network, graph, or statistical-analysis
dependency, keeping 'MiscMetabar' lean. Built on top of 'MiscMetabar'.
