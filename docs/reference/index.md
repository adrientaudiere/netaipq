# Package index

## Networks

Build and visualise association and co-occurrence networks of the taxa
or samples of a phyloseq object.

- [`bipartite_network_pq()`](https://adrientaudiere.github.io/netaipq/reference/bipartite_network_pq.md)
  : Bipartite network of samples and taxa from a phyloseq object
- [`cooccurrence_network_pq()`](https://adrientaudiere.github.io/netaipq/reference/cooccurrence_network_pq.md)
  : Probabilistic species co-occurrence from a phyloseq object
- [`ggclusternet_pq()`](https://adrientaudiere.github.io/netaipq/reference/ggclusternet_pq.md)
  : Microbial correlation network of a phyloseq object using
  ggClusterNet
- [`netassoc_network_pq()`](https://adrientaudiere.github.io/netaipq/reference/netassoc_network_pq.md)
  : Species-association network of a phyloseq object using netassoc

## Causal inference

Confront hypothesised causal models with phyloseq sample data.

- [`dag_test_pq()`](https://adrientaudiere.github.io/netaipq/reference/dag_test_pq.md)
  : Test a causal DAG against phyloseq sample data

## Compositional correlation

Relate taxa to environmental variables while accounting for
compositionality.

- [`sparcev_pq()`](https://adrientaudiere.github.io/netaipq/reference/sparcev_pq.md)
  : Compositional taxa-environment correlation with SparCEV
